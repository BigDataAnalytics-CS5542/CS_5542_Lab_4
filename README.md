## Setup Instructions

1. Clone repository:

git clone <repo-url>
cd CS_5542_LAB_4

2. Create virtual environment:

python3 -m venv venv

3. Activate virtual environment:

Mac/Linux:
source venv/bin/activate

Windows:
venv\Scripts\activate

4. Install dependencies:

pip install -r requirements.txt

5. Run Streamlit app:

streamlit run ui/app.py
